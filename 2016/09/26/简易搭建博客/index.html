免费简易搭建Hexo博客网站

2016-09-26
#Node.js

经过一个下午的折腾，我还是终于把自己的一个静态页面博客搭起来了，走了好多弯路~自学成才的路是坎坷的~如果不记录下来，我一个下午的折腾过几天估计就忘了，进入正题！

工具及环境准备工作

  • Mac OSX EI Capitan 电脑
  • atom 文本编辑器，或任意一款文本编辑器
  • github 账号
  • git 软件，Mac用户自带有
  • hexo模块，查看官方文档

首先要做的就是进入hexo文档了解Hexo到底是什么东西

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

一、安装hexo

安装需要两个环境

  • Node.js
  • Git

因为Mac自带有git，我就不介绍git的安装了

安装Node.js

安装 Node.js 的最佳方式是使用 nvm。

cURL:

$ curl https://raw.github.com/creationix/nvm/master/install.sh | sh

Wget:

$ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh

安装完成后，重启终端并执行下列命令即可安装 Node.js。

$ nvm install stable

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

$ npm install -g hexo-cli

二、建立网站

安装完成Hexo后，选择一个文件夹作为建立网站的存放的路径比如我选择的是桌面Blog文件夹

$ cd desktop/blog

初始化网站

$ hexo init

执行下面的命令，系统会可以根据package.json文件中dependencies的配置安装所有依赖包

$ npm install

命令不熟悉的，一定要去查官方文档，这句话不管是什么样的程序员都管用！文档就是教程，传送在上面

新建完成之后，根目录文件夹下各个目录的含义，可以参考Hexo官网的解释：目录信息

三、启动网站

在建立网站的目录下使用以下命令，之后访问自己的host http://localhost:4000/
可以看到一个默认的博客模板,Hello，world博文

$ hexo server

每次我们写完博客后，先到本地预览，然后部署到github.page上，这样会方便很多

四、配置部署

首先申请github账号，然后建立仓库 ，如下图

填写产库名字仓库名字必须是yourname.github.io 这个格式(不确定，反正不是这个名字我试了几次不行，没找到原因 = 。=)，yourname 是需要你自己起的名字，打开仓库复制Https

用ATOM打开hexo安装博客模块下的_config.yml 文件,搜索deploy，修改成如下

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: https://github.com/yourname/yourname.github.io.git
  branch: master

repository:参数后面就是你复制自己仓库https连接

:后面一定要空格。

安装Git包，执行以下命令

$ npm install hexo-deployer-git --save

部署

$ hexo d

现在可以通过访问 https://yourname.github.io/ 访问自己的博客啦。如果不知道自己的博客连接，进入github自己的仓库，点击右上角Setting 拖到下面 github page，里面有个你自己的博客地址。
